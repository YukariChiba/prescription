[点击下崽](https://raw.githubusercontent.com/YukariChiba/prescription/main/main.pdf)

## 如何构建

使用 `xelatex` 构建 `main.tex`。

### 修改内容：

修改 `macro.tex` 中的对应字段。

### 修改药品：

修改 `medicine.tex` 中的对应字段。
